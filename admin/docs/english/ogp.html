Geeklog OGP(Open Graph Protocol) plugin

Configuration

OGP Main Settings

Variable Default Value Description
Your Facebook User ID(s) (none) Enter your Facebook user ID.
Your Facebook Application ID (none) Enter your Facebook Application ID(App ID). You have to set this item if you want to use like buttons and/or Facebook comments supported by the plugin.
Your site's default image site_URL/admin/plugins/ogp/images/facebook_128.png Enter the URL to the default image to display when you have no image to show in your articles. The image must be larger than 50*50 pixels.

Facebook Like button

Variable Default Value Description
Show "send" button Yes Setting this item to Yes shows "send" button.
Show faces Yes Setting this item to Yes shows profile images of the users who click on "Like" button.
Style layout Standard Selecte the layout of Like button you prefer.
Width (pixels) 450 Set the width of Like button widget.
Action Like Choose between "Like" and "Recommend".
Color Scheme Light Choose between "Light" and "Dark".
Font Default Choose the font you prefer.

Facebook Comments

Variable Default Value Description
Number of posts 10 Enter the maximum number of comments yo show.
Width (pixels) 500 Set the width of Facebook comments widget.
Color Scheme Light Choose between "Light" and "Dark".
