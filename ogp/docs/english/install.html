Install/Uninstall instruction for the Geeklog OGP(Open Graph Protocol) plugin

In the following descriptions

  • <geeklog_dir> is the directory where the system config.php file resides
  • <admin> is the directory where the administration files reside (usually, under <public_html>)

INSTALL:

  1. Uncompress the ogp plugin archive while in the <geeklog_dir>/plugins directory. The archive will create a directory called ogp in the plugins directory.
  2. Create the admin directory. Under your <admin>/plugins/ directory create a directory called ogp.
  3. Change to your <geeklog_dir>/plugins/ogp/ directory. Copy the files in the admin directory to the <admin>/plugins/ogp/ directory your created in step 2.
  4. Change <public_html>/layout/your_theme/header.thtml.
    (before) <html{html_attribute}{xmlns}>
    (after) <html{html_attribute}{xmlns} xmlns:og="http://ogp.me/ns#">
  5. Log in to your Geeklog as a root user and click on "Plugins", then hit "intall" button. The OGP plugin should now be installed and functioning.
  6. IMPORTANT: Go to Configuration Panel and set your Facebook User ID(s). If you don't know your Facebook User ID, then use This App.

UNINSTALL:

  1. Log in to your Geeklog as a root user and click on "Plugins", then hit "unintall" button.
  2. Delete the two plugin directories created in the install process: <geeklog-dir>/plugins/ogp/ and /ogp/ and <admin>/plugins/ogp/

That's it.
